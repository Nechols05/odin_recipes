# Odin Recipes

This is my first project from **The Odin Project** Foundations course.  
It contains a simple website with multiple recipes, built using basic HTML.

## Features
- Main index page linking to individual recipes
- Separate HTML pages for each recipe
- Basic HTML structure (headings, lists, images, links)

## Purpose
This project helps me practice:
- Creating HTML files
- Using lists and images
- Linking between pages
- Organizing a project folder

## How to View
Open `index.html` in a browser to see the main page.
